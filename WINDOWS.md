# 🪟 Windows & MS-DOS: True Native Power!

LibScript doesn't treat Windows as an afterthought. We are bringing true, native, zero-dependency provisioning to Windows, making LibScript the ultimate **VERY VERY cross-platform** orchestrator!

## ⚡ The Ultimate WAMP Stack Maker!
Forget clunky third-party GUIs. With LibScript, you can spin up a pristine **WAMP (Windows, Apache, MySQL, PHP) stack** directly from your command prompt! It installs natively, configures the environment variables, and registers the Windows Services automatically!

## 📦 Generating Nice Windows Installers!
This is where LibScript absolutely shines! If you have built a generic stack of tools and want to distribute it to Windows users, you don't need to learn WiX or NSIS.

LibScript's `package_as` command will automatically compile your stack into:
- **.MSI Installers** (via WiX)
- **.EXE Installers** (via InnoSetup or NSIS)

It bundles the dependencies, sets up the UI, and creates a highly professional installer dynamically!

## 🦕 Legacy MS-DOS Support
We even support legacy `.bat` files for bootstrapping on raw, vintage MS-DOS setups, ensuring LibScript can run absolutely anywhere!
