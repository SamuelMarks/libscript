Nuget
=====

Bootstrap module for the `nuget` package manager.

## Usage
Ensures the `nuget` executable is available. Uses system package manager mapping (e.g., `nuget` package on Debian).

## Platform Support
- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Supported.

## Variables
See `vars.schema.json` for details on available variables.
