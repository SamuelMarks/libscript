Cargo Binstall
==============

Bootstrap module for the `cargo-binstall` package manager.

## Usage
Ensures the `cargo-binstall` executable is available. This relies on the core language toolchain appropriate for the tool.

## Platform Support
- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Supported.

## Variables
See `vars.schema.json` for details on available variables.
