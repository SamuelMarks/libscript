# pyenv

Bootstrap module for `pyenv` (Python Version Manager).

## Overview

Ensures the `pyenv` executable/shell function is available. It installs `pyenv` on UNIX-like systems, and `pyenv-win` on Windows.

## OS Compatibility

- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Supported.

## Variables

See `vars.schema.json` for details on available variables.
