Pnpm
====

Bootstrap module for the `pnpm` package manager.

## Usage
Ensures the `pnpm` executable is available by pulling in Node.js/npm and running `npm install -g pnpm`.

## Platform Support
- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Supported.

## Variables
See `vars.schema.json` for details on available variables.
