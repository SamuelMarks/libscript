# pipx

Bootstrap module for the `pipx` package manager.

## Overview

Ensures the `pipx` executable is available for installing Python CLI tools in isolated environments. This module relies on the core Python toolchain and `pip`.

## OS Compatibility

- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Supported.

See `vars.schema.json` for details on available variables.
