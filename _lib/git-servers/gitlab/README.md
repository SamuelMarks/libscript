GitLab
======

GitLab Community Edition (CE) is a self-hosted Git management software based on Ruby on Rails.

**Note:** GitLab is a very large application stack. This module attempts to install it via the official system package manager (Omnibus). If you encounter issues, consider using Docker instead.

## Usage
```sh
libscript install gitlab latest
```

## Options
See `vars.schema.json` for all available configuration options.

## Platform Support
- Linux
- macOS
- Windows
