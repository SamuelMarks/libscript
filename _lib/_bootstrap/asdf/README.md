# asdf

Bootstrap module for the `asdf` universal version manager.

## Overview

Clones and installs `asdf` to `~/.asdf`. Requires manual sourcing in bash/zsh profiles by the user for persistence.

## OS Compatibility

- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Not supported (requires WSL or MSYS).

## Variables

See `vars.schema.json` for details on available variables.
