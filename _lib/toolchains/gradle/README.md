Gradle
======

Bootstrap module for `gradle` (Java / JVM build automation system).

## Usage
Ensures the `gradle` executable is available. Relies on the core Java toolchain and commonly uses `sdkman` to manage versions on UNIX-like environments.

## Platform Support
- **Linux**: Supported.
- **macOS**: Supported.
- **Windows**: Supported.

## Variables
See `vars.schema.json` for details on available variables.
